# 2.8 – List View và History

## 1. Mục tiêu bài học

- Sau bài này, học viên sẽ:
  - Biết **List View** dùng để làm gì và khi nào cần dùng.
  - Biết cách dùng **History** (lịch sử thao tác) để undo/redo hiệu quả.
  - Biết cách xem danh sách hotspot toàn node bằng List View.

## 2. List View

### 2.1. List View là gì?

- **List View** là chế độ hiển thị dạng bảng/danh sách cho **hotspot** của node đang chọn.
- Thay vì click từng hotspot trên Viewer, List View cho phép:
  - **Xem nhanh** tất cả hotspot: ID, tên, loại, node đích, tọa độ.
  - **Chọn nhiều hotspot** cùng lúc.
  - **Chỉnh sửa hàng loạt** (bulk edit) thông số.
  - **Sort / lọc** hotspot theo cột.

Mở List View: tab **List** ở phần dưới cùng của Viewer, hoặc **View → List View**.

Docs: `https://ggnome.com/doc/pano2vr/workspace/list-view/`

### 2.2. Cột trong List View (hotspots)

| Cột | Mô tả |
|---|---|
| ID | Node ID của hotspot |
| Title | Tên hotspot (từ User Data) |
| Type | Point / Polygon |
| Target | Node đích (khi click sẽ move tới) |
| Action | Loại action (go to node, open URL, toggle info,…) |
| Pan / Tilt | Tọa độ tâm hotspot trên panorama |
| Visible | Bật/tắt hiển thị hotspot |

### 2.3. Khi nào cần dùng List View?

| Tình huống | Lợi ích |
|---|---|
| Tour có 50+ hotspot | Tìm nhanh không cần scroll Viewer |
| Cần đổi node đích hàng loạt | Select nhiều hotspot → bulk-change Target |
| Kiểm tra hotspot bị ẩn (Visible = off) | Dễ thấy qua cột Visible |
| Audit tất cả action hotspot | Nhìn tổng quát không bỏ sót |

## 3. History (Lịch sử thao tác)

### 3.1. History là gì?

- **History panel** lưu danh sách tất cả hành động vừa thực hiện trong phiên làm việc hiện tại.
- Cho phép **Undo** (`Ctrl+Z`) từng bước và **Redo** (`Ctrl+Y`) lại.
- Click vào một mục trong History để nhảy thẳng về trạng thái đó (multi-level undo).

Mở History: **Window → History** hoặc tab **History** (nếu có sẵn trong layout).

### 3.2. Giới hạn History

- History **không được lưu** cùng file `.p2vr`. Khi đóng/ mở lại project, lịch sử bị xoá.
- Số bước undo lưu được phụ thuộc vào cài đặt (thường 50–100 bước).
- **Không phải mọi thao tác đều Undo được**: VD, sau khi Generate Output, không undo được output đã xuất ra ổ đĩa.

### 3.3. Thao tác thực tế với History

1. **Multi-step undo**: nhấn `Ctrl+Z` nhiều lần hoặc click vào bước cụ thể trong History panel.
2. **Redo**: `Ctrl+Y` để làm lại bước đã undo.
3. **Save trước khi thử nghiệm lớn**: vì History bị xoá khi tắt Pano2VR, hãy `Ctrl+S` trước mỗi lần thử nghiệm lớn (đặt nhiều hotspot, thay đổi skin,…).

## 4. Lưu ý

- **List View vs. Tour Browser**: Tour Browser quản lý **node**, List View quản lý **hotspot** bên trong 1 node.
- **Chọn nhiều hotspot** trong List View: `Ctrl+Click` hoặc `Shift+Click`.
- History và List View đều là công cụ hỗ trợ – không ảnh hưởng đến output xuất ra.
- Với tour lớn, nên **backup file `.p2vr`** (copy sang folder Backup) định kỳ, không chỉ dựa vào Undo.

## 6. Gợi ý bài tập

1. Mở 1 node có nhiều hotspot. Mở List View, thử click vào từng cột để sort.
2. Chọn 3 hotspot cùng lúc trong List View → thay đổi `Visible = off` cho cả 3 cùng lúc.
3. Thực hiện 5 thao tác (thêm hotspot, đổi title,…), sau đó mở History và undo từng bước. Quan sát sự thay đổi.
