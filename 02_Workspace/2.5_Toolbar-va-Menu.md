# 2.5 – Toolbar và Menu

## 1. Mục tiêu bài học

- Sau bài này, học viên sẽ:
  - Nhận diện được các nút quan trọng trên **Main Toolbar**.
  - Biết cách truy cập các chức năng thường dùng qua **Menu bar**.
  - Nhớ các **phím tắt** (keyboard shortcuts) quan trọng nhất để tăng tốc độ làm việc.

## 2. Main Toolbar

Main Toolbar nằm ở **phía trên** giao diện, chứa các nút truy cập nhanh đến chức năng chính.

Docs: `https://ggnome.com/doc/pano2vr/workspace/toolbar/`

### 2.1. Nhóm File/Project

| Biểu tượng | Chức năng | Phím tắt |
|---|---|---|
| New | Tạo project mới | `Ctrl+N` |
| Open | Mở project (`.p2vr`) | `Ctrl+O` |
| Save | Lưu project | `Ctrl+S` |
| Save As | Lưu thành tên khác | `Ctrl+Shift+S` |

### 2.2. Nhóm Input/Node

| Biểu tượng | Chức năng |
|---|---|
| Add Input | Thêm ảnh/video 360° vào tour |
| Tour Browser | Toggle panel Tour Browser |
| Previous / Next Node | Chuyển qua node trước/sau |

### 2.3. Nhóm Output

| Biểu tượng | Chức năng |
|---|---|
| Output | Toggle Output Panel |
| Generate Output | Xuất tất cả output đang bật |

### 2.4. Nhóm View/Preview

| Biểu tượng | Chức năng | Phím tắt |
|---|---|---|
| Preview | Bật/tắt Preview mode | `P` |
| Viewer Mode | Dropdown chọn mode hiển thị | `Ctrl+1/2/3` |

### 2.5. Nút Skin Editor

| Biểu tượng | Chức năng |
|---|---|
| Skin Editor | Mở Skin Editor để thiết kế giao diện người dùng |

## 3. Menu Bar

### 3.1. Menu File

| Mục | Chức năng |
|---|---|
| New / Open / Save / Save As | Quản lý project |
| Import | Import node từ nguồn khác |
| Export | Xuất project (ngoài luồng Output Panel) |
| Find and Relocate Missing Files | Tìm & cập nhật đường dẫn file bị đổi tên/ di chuyển |
| Recent Projects | Mở nhanh project gần đây |

### 3.2. Menu Edit

| Mục | Chức năng |
|---|---|
| Undo / Redo | Hoàn tác / làm lại | 
| Set Initial View | Lưu góc nhìn hiện tại làm Initial View |
| Copy / Paste Node Settings | Sao chép cài đặt node (hotspot, user data…) sang node khác |
| Project Settings | Cài đặt chung cho cả project |

### 3.3. Menu View

| Mục | Chức năng |
|---|---|
| Viewer Mode | Chọn mode hiển thị (Normal, Equirectangular, Fisheye,…) |
| Show Hotspots | Toggle hiển thị hotspot trong Viewer |
| Show Grid | Toggle lưới kinh/vĩ độ lên Viewer |
| Panels | Ẩn/hiện các panel (Tour Browser, Properties, Output, History,…) |

### 3.4. Menu Help

| Mục | Chức năng |
|---|---|
| Pano2VR Help | Mở docs online |
| Check for Updates | Kiểm tra phiên bản mới |
| License | Quản lý key bản quyền |
| About | Thông tin version đang dùng |

## 4. Phím tắt quan trọng

| Phím tắt | Chức năng |
|---|---|
| `Ctrl+S` | Lưu |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `I` | Đặt Initial View từ góc nhìn hiện tại |
| `P` | Toggle Preview mode |
| `H` | Add Point Hotspot tại tâm Viewer |
| `Ctrl+]` | Node tiếp theo |
| `Ctrl+[` | Node trước đó |
| `Ctrl+1` | Viewer mode: Normal |
| `Ctrl+2` | Viewer mode: Equirectangular |
| `Ctrl+3` | Viewer mode: Fisheye |
| `Delete` | Xoá hotspot đang chọn |
| `F5` | Refresh output preview |

> Tip: Mở **Help → Keyboard Shortcuts** trong phần mềm để xem danh sách đầy đủ cho phiên bản đang dùng.

## 5. Lưu ý thực tế

- Một số nút trên Toolbar **thay đổi ngữ cảnh** tuỳ panel đang focus (VD: khi Skin Editor đang mở, Toolbar sẽ hiện thêm công cụ Skin).
- **Save thường xuyên** – Pano2VR không có autosave (tính đến 7.1). `Ctrl+S` phải trở thành phản xạ.
- Có thể **tùy chỉnh layout** toolbar (kéo panel, ghim) nhưng thường không cần thiết với workflow thông thường.

## 6. Tài liệu tham khảo

- Toolbar – `https://ggnome.com/doc/pano2vr/workspace/toolbar/`
- Keyboard Shortcuts – `https://ggnome.com/doc/pano2vr/reference/keyboard-shortcuts/`

## 7. Gợi ý bài tập

1. Mở Pano2VR, thử lần lượt các phím tắt trong bảng trên – ghi lại 5 phím tắt bạn dùng thường xuyên nhất.
2. Tìm trong menu **Edit → Copy Node Settings** → thử copy settings từ node A sang node B trong tour.
3. Dùng **View → Show Grid** để bật lưới kinh/vĩ độ – quan sát sự thay đổi trong Viewer.
