# 2.9 – Settings & Preferences

## 1. Mục tiêu bài học

- Sau bài này, học viên sẽ:
  - Biết truy cập **Preferences (Tùy chọn)** của Pano2VR.
  - Cấu hình được các tùy chọn thường dùng: ngôn ngữ, thư mục mặc định, tile caching, preview browser.
  - Phân biệt **Preferences** (cài đặt phần mềm) vs. **Project Settings** (cài đặt project).

## 2. Truy cập Preferences

- **Windows**: menu **Edit → Preferences** (hoặc `Ctrl+,`).
- **macOS**: menu **Pano2VR → Preferences**.

Docs: `https://ggnome.com/doc/pano2vr/workspace/preferences/`

## 3. Các nhóm tùy chọn quan trọng

### 3.1. General

| Tùy chọn | Mô tả | Khuyến nghị |
|---|---|---|
| Language | Ngôn ngữ giao diện phần mềm | English (tài liệu gốc Ggnome dùng tiếng Anh) |
| Check for updates | Tự kiểm tra update khi khởi động | Bật |
| Undo steps | Số bước Undo lưu được | 50–100 |
| Theme | Giao diện tối/sáng | Tuỳ sở thích |

### 3.2. Paths (Đường dẫn)

| Tùy chọn | Mô tả |
|---|---|
| Default output folder | Thư mục mặc định để xuất output |
| Skin folder | Nơi Pano2VR tìm kiếm file `.ggsk` |
| Temp folder | Thư mục lưu file tạm (caching, preview) |

> Nên đặt **Temp folder** trên ổ SSD có dung lượng trống lớn (≥10 GB) để Pano2VR caching tile nhanh hơn. Tránh đặt trên ổ mạng (NAS/network drive).

### 3.3. Preview

| Tùy chọn | Mô tả |
|---|---|
| Browser | Trình duyệt dùng để preview output Web |
| Local server port | Cổng của local web server (mặc định: 8080) |

- Pano2VR tự tích hợp **local web server nhỏ** để preview – tránh lỗi CORS khi mở `file://`.
- Nếu cổng 8080 bị chiếm bởi ứng dụng khác, đổi sang 8081, 8888,…

### 3.4. Tiles & Caching

| Tùy chọn | Mô tả |
|---|---|
| Tile cache size | Dung lượng cache tile tối đa | 
| Clear cache | Xoá cache cũ để giải phóng ổ đĩa |
| Multithreading | Số luồng CPU dùng để render/tile | 

> Khi máy bị chậm trong quá trình Generate Output, thử tăng số luồng CPU trong Multithreading (thường = số CPU core -1).

### 3.5. Skin Editor

| Tùy chọn | Mô tả |
|---|---|
| Default font | Font chữ mặc định khi tạo text element trong Skin |
| Grid size | Khoảng cách lưới trong Skin Editor canvas |
| Snap to grid | Tự động căn chỉnh vào lưới |

## 4. Preferences vs. Project Settings

| | Preferences | Project Settings |
|---|---|---|
| Phạm vi | Toàn bộ phần mềm (mọi project) | Chỉ project hiện tại |
| Lưu ở đâu | File cấu hình của phần mềm | Trong file `.p2vr` |
| Truy cập | Edit → Preferences | Edit → Project Settings |
| Ví dụ | Preview browser, temp folder | Start node, skin mặc định, author |

## 5. Lưu ý thực tế

- **Reset Preferences**: nếu Pano2VR hành xử lạ sau update, thử Reset Preferences về default (có nút trong Preferences dialog).
- **Di chuyển máy**: khi chuyển sang máy mới, export Preferences hoặc ghi lại cài đặt quan trọng (đường dẫn, cổng preview) để cấu hình lại nhanh.
- **Skin folder**: nếu bạn lưu skin custom ở folder chung của team, khai báo đường dẫn đó trong Skin folder → Pano2VR tự nhận diện.

## 6. Tài liệu tham khảo

- Preferences – `https://ggnome.com/doc/pano2vr/workspace/preferences/`
- Project Settings – `https://ggnome.com/doc/pano2vr/reference/project-settings/`

## 7. Gợi ý bài tập

1. Mở Preferences → đổi **Preview browser** thành trình duyệt bạn hay dùng nhất.
2. Đặt **Default output folder** về Desktop/test để dễ kiểm tra.
3. Thử **Clear cache** trong phần Tiles – quan sát dung lượng ổ đĩa trước và sau.
