# 2.7 – Output Panel

## 1. Mục tiêu bài học

- Sau bài này, học viên sẽ:
  - Biết **Output Panel là gì** và cách thêm / bật / tắt các output.
  - Phân biệt được các **Output Type** chính.
  - Thực hiện được bước xuất output đầu tiên (Web/HTML5).
  - Biết cách mở kết quả ngay sau khi xuất.

## 2. Output Panel là gì?

- **Output Panel** là nơi cấu hình và quản lý tất cả các định dạng xuất file của project.
- Mỗi project có thể có **nhiều output** cùng lúc (VD: vừa xuất Web, vừa xuất Transformation ảnh tĩnh).
- Nằm ở phía dưới giao diện (mặc định) hoặc mở từ Toolbar.

Docs: `https://ggnome.com/doc/pano2vr/output/`

## 3. Giao diện Output Panel

- **Danh sách output**: liệt kê các output đã thêm vào project (mỗi item = 1 cấu hình xuất).
- **Checkbox** trước mỗi output: bật/tắt output đó khi nhấn Generate.
- **Output type badge**: chỉ loại output (Web, Transformation, Animation, VR, Package, Cloud).
- Nút **Generate Output** (hoặc Toolbar): xuất tất cả output đang bật.
- Nút **+**: thêm output mới.

## 4. Các Output Type

### 4.1. Web (HTML5) – Quan trọng nhất

- Xuất tour thành **website HTML5** (kèm JS, CSS, tiles) chạy trên trình duyệt.
- Không cần plugin – chạy trên mọi thiết bị hiện đại (PC, tablet, mobile).
- Tham số chính:
  - **Output folder**: thư mục lưu file xuất.
  - **HTML file name**: tên file `index.html`.
  - **Tiles**: cài đặt tiling (kích thước tile, mức zoom – liên quan đến chất lượng & hiệu năng tải).
  - **Skin**: chọn skin file (`.ggsk`) áp dụng cho tour.
  - **Include Tour Browser data**: xuất kèm dữ liệu tour (tên node, GPS,…).

### 4.2. Transformation

- Xuất ảnh/video tĩnh từ panorama theo **góc nhìn cố định** hoặc theo lưới góc.
- VD: xuất một tập 24 ảnh phẳng từ panorama (mỗi ảnh 90° FoV, xoay mỗi 15°) → dùng trong VR headset, marketing,…
- Hữu ích cho: Tiny Planet, grid ảnh, Still image export.

### 4.3. Animation

- Xuất **video** (`.mp4`, `.gif`,…) là đoạn flythrough tự động qua các node.
- Kiểm soát được: keyframe, tốc độ, transition.

### 4.4. VR (WebXR)

- Xuất tour hỗ trợ **VR headset** qua WebXR API.
- Chạy trên trình duyệt hỗ trợ WebXR (Chrome, Firefox Reality, Meta Browser).
- Cần HTTPS để WebXR hoạt động.

### 4.5. Package / CMS

- Đóng gói tour thành định dạng phù hợp với một số **CMS hoặc platform** cụ thể.
- VD: xuất dạng `.zip` để upload lên hosting, hoặc dạng phù hợp với Krpano server, iframe embed,…

### 4.6. Cloud (Ggnome Cloud)

- Upload trực tiếp lên **cloud hosting của Ggnome** (pano2vr.cloud).
- Không cần cấu hình hosting – nhanh nhất để share demo cho client.
- Yêu cầu tài khoản Ggnome.

## 5. Quy trình xuất Web cơ bản (lần đầu)

1. Trong Output Panel, nhấn **+** → chọn **Web**.
2. Cài đặt **Output folder**: chọn thư mục trống (VD: `exports/my-tour`).
3. Cài đặt **Skin**: chọn skin mặc định hoặc skin đã tạo.
4. Để các tham số Tiles theo mặc định.
5. Đảm bảo checkbox của output Web được bật.
6. Nhấn **Generate Output** (nút trên Toolbar hoặc trong Output Panel).
7. Sau khi xong: nhấn **Preview** (biểu tượng mắt bên cạnh output) hoặc mở `index.html` bằng trình duyệt.

> Lưu ý: Nếu mở `index.html` trực tiếp bằng double-click (file://) sẽ gặp lỗi CORS. Phải dùng local web server hoặc tính năng Preview của Pano2VR. Xem chi tiết ở bài 3.x.

## 6. Lưu ý thực tế

- **Tên output folder không dấu, không space**: tránh lỗi đường dẫn khi upload.
- **Overwrite**: nếu xuất lại vào cùng folder cũ, các file sẽ bị ghi đè – an toàn để update.
- **Tiles vs. Single file**: với tour ảnh lớn (>8000px), dùng Tiling để tối ưu tải – trình duyệt chỉ tải tile đang xem, không tải cả ảnh gốc.
- **Multiple outputs**: có thể thêm nhiều output Web với cấu hình khác nhau (VD: 1 bản HQ và 1 bản LQ để test) và chọn xuất cái nào tuỳ.

## 8. Gợi ý bài tập

1. Thêm **Web output** cho project hiện tại. Xuất vào folder `exports/test-tour`. Mở bằng Preview để xem kết quả.
2. Thêm thêm 1 **Transformation output** với 1 góc nhìn cố định (VD: Front, FoV 90°). Xuất và kiểm tra file ảnh tĩnh được tạo ra.
3. Thử tắt checkbox của Web output → nhấn Generate → xác nhận Transformation output vẫn chạy nhưng Web output không chạy.
