# 2.6 – Properties Panel

## 1. Mục tiêu bài học

- Sau bài này, học viên sẽ:
  - Biết cấu trúc của **Properties Panel** và các tab chính.
  - Cài đặt được **Node Properties** (title, description, north offset, initial view, GPS).
  - Cài đặt được **Viewer Properties** (FoV limits, auto-rotate, gyroscope).
  - Biết cài đặt **Project Properties** ảnh hưởng tới toàn bộ tour.

## 2. Properties Panel là gì?

- **Properties Panel** nằm ở bên phải Viewer (mặc định).
- Đây là nơi cấu hình **tất cả thông số** của node đang được chọn, của Viewer, và của project.
- Gồm nhiều tab/ nhóm, mỗi nhóm quản lý một khía cạnh khác nhau.

Docs: `https://ggnome.com/doc/pano2vr/workspace/properties/`

## 3. Các nhóm chính trong Properties

### 3.1. Node – Input

| Tham số | Mô tả |
|---|---|
| Input file | Đường dẫn file ảnh/video gốc |
| Input type | Loại input: Equirectangular, Cubemap, Fisheye, Flat,… |
| Projection | Loại projection nếu input không phải equirectangular |
| Crop / Rotate | Xoay, crop ảnh nếu cần chỉnh hướng |

> Nếu input là **fisheye** hay **cubemap** từ camera, phải khai báo đúng Input Type để Pano2VR nhận diện và chuyển đổi nội bộ sang equirectangular.

### 3.2. Node – Initial View

| Tham số | Đơn vị | Mô tả |
|---|---|---|
| Pan | độ | Hướng ngang ban đầu |
| Tilt | độ | Góc ngẩng/cúi ban đầu |
| FoV | độ | Góc nhìn ban đầu (thường 90°) |

- Có thể nhập thủ công hoặc dùng phím `I` trong Viewer để lấy góc hiện tại.

### 3.3. Node – User Data

| Tham số | Mô tả |
|---|---|
| Title | Tên node (dùng trong menu tour, tooltip,…) |
| Description | Mô tả – có thể dùng trong hotspot info/ tour map popup |
| Thumbnail | Ảnh thumbnail tuỳ chọn (nếu không muốn dùng auto-generated) |
| GPS | Lat/Lon cho Tour Map và geo-tagging |

> `Title` là trường quan trọng nhất – cần điền đầy đủ cho tất cả node từ đầu. Sau này skin tự động lấy Title để hiển thị tên phòng / điểm tham quan.

### 3.4. Node – North Offset

- Dịch chuyển 0° (hướng "North" trong file ảnh) theo thực tế địa lý.
- Nhập số độ chênh lệch (VD: `+45` nghĩa là North thực tế lệch 45° so với North trong ảnh).
- Quan trọng cho Tour Map & compass orientation.

### 3.5. Node – Patch

- Xem danh sách các **patch** (vùng vá) đang áp dụng cho node.
- Có thể điều chỉnh độ mờ (opacity) của patch overlay.
- Chi tiết về Patching sẽ học ở Phần 3.

### 3.6. Viewer Settings

| Tham số | Mô tả |
|---|---|
| FoV min / max | Giới hạn FoV tối thiểu / tối đa người dùng có thể zoom |
| Pan limits | Giới hạn pan ngang (bảo vệ vùng nhất định không xem được) |
| Tilt limits | Giới hạn tilt (VD: không cho xem zenith) |
| Auto-rotate | Tự động xoay panorama khi mở – tốc độ và chiều |
| Auto-rotate speed | Tốc độ (độ/giây) |
| Gyroscope | Bật/tắt điều hướng bằng con quay hồi chuyển (di động) |

> **FoV limits** thực tế: đặt min 40° và max 120° là phổ biến. Tránh để FoV max > 120° vì gây méo ảnh khó chịu.

### 3.7. Project Settings

Truy cập: **Edit → Project Settings** hoặc tab **Project** trong Properties (nếu không có node nào được chọn).

| Tham số | Mô tả |
|---|---|
| Start Node | Node bắt đầu mặc định toàn tour |
| Background color | Màu nền hiển thị trong lúc load |
| Skin file | Skin (.ggsk) áp dụng cho toàn project |
| Output folder | Thư mục mặc định để xuất file |
| Author / Copyright | Thông tin metadata cho output |

## 4. Quy trình điền Properties tiêu chuẩn

Sau khi import node, thực hiện theo thứ tự:

1. `Node → Input Type`: xác nhận đúng loại (thường = Equirectangular).
2. `Node → User Data → Title`: đặt tên rõ ràng (VD: "Phòng khách", "Tầng 1 – Sảnh đón").
3. `Node → GPS`: nhập tọa độ nếu có Tour Map.
4. `Node → North Offset`: căn hướng nếu cần.
5. Viewer → di chuyển đến góc đẹp → nhấn `I` → `Initial View` tự cập nhật.
6. `Viewer Settings → Auto-rotate`: bật nếu client muốn.

## 5. Lưu ý thực tế

- **Sao chép settings sang nhiều node**: Edit → Copy Node Settings → chọn tham số muốn copy → Paste to Selected Nodes. Rất hữu ích khi cần đồng bộ FoV limits, auto-rotate cho cả tour.
- **GPS accuracy**: tọa độ GPS từ camera thường lệch 10–50m. Với Tour Map dạng bản đồ vệ tinh, cần kiểm tra lại thủ công.
- **Title dài**: nếu Title quá dài, menu skin có thể bị tràn. Nên giới hạn ~30 ký tự cho Title.

## 7. Gợi ý bài tập

1. Mở 1 project có ít nhất 3 node. Điền đầy đủ Title và Description cho từng node.
2. Bật Auto-rotate với tốc độ 10°/s. Chuyển sang Preview mode – quan sát.
3. Đặt FoV min = 40°, max = 110°. Thử scroll zoom trong Preview mode để xác nhận giới hạn.
4. Dùng **Copy Node Settings → Viewer Settings** → paste vào 2 node còn lại để đồng bộ cài đặt.
