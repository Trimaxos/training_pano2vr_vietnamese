# 7.11 – Web Element, PDF Hotspot và Custom Properties

## 1. Mục tiêu bài học

- Sau bài này, học viên sẽ:
  - Hiểu các tính năng mở rộng mới đáng chú ý từ Pano2VR 7.1.
  - Biết khi nào nên dùng `PDF Hotspot Type`, `Web Element`, và `Custom Properties`.
  - Có mốc video chính xác để tham khảo nhanh.

## 2. PDF Hotspot Type

- `PDF Hotspot Type` phù hợp khi tour cần mở tài liệu PDF trực tiếp từ hotspot.
- Use case:
  - Catalogue sản phẩm.
  - Hướng dẫn kỹ thuật.
  - Hồ sơ bàn giao hoặc bản vẽ đính kèm.

## 3. Web Element

- `Web Element` cho phép mở rộng lớp giao diện với nội dung web nhúng hoặc logic web mạnh hơn.
- Phù hợp khi:
  - Cần hiển thị nội dung web động.
  - Cần tích hợp thành phần ngoài skin cơ bản.

## 4. Custom Properties

- `Custom Properties` hữu ích khi dự án cần thêm metadata riêng.
- Phù hợp cho:
  - workflow nội bộ,
  - ghi chú kỹ thuật,
  - hoặc logic tùy biến về sau.

## 5. Khi nào đáng dùng?

| Tính năng | Khi nên dùng |
|---|---|
| PDF Hotspot | Dự án cần đính tài liệu tra cứu |
| Web Element | Dự án cần tích hợp UI/web content nâng cao |
| Custom Properties | Dự án cần metadata tùy biến hoặc logic kỹ thuật riêng |
