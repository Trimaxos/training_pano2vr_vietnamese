# 7.10 – Mở rộng tour bằng JavaScript API

## 1. Mục tiêu bài học

- Sau bài này, học viên sẽ:
  - Hiểu khi nào Pano2VR cần `JavaScript API`.
  - Biết các hướng tích hợp thường gặp giữa tour và trang web xung quanh.
  - Có roadmap học video đúng theo chapter để nghiên cứu sâu hơn.

## 2. Khi nào cần JavaScript API?

- Khi skin mặc định không đủ.
- Khi tour cần giao tiếp với:
  - trang HTML bên ngoài,
  - hệ thống web của công ty,
  - nút điều khiển riêng ngoài skin,
  - logic tùy biến theo dữ liệu động.

## 3. Các tình huống phổ biến

| Tình huống | API giúp gì |
|---|---|
| Nút ngoài web điều khiển tour | Gọi chuyển node, đổi view |
| Đồng bộ UI bên ngoài | Cập nhật trạng thái theo node hiện tại |
| Truyền dữ liệu vào skin | Set biến từ trang ngoài |
| Link thẳng tới một view cụ thể | Tạo trải nghiệm điều hướng có chủ đích |
