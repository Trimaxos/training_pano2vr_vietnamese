# 7.7 – Mẹo và tham chiếu nâng cao

## 1. Mục tiêu bài học

- Sau bài này, học viên sẽ:
  - Biết các hướng mở rộng nâng cao sau khi đã nắm workflow chuẩn.
  - Nhận diện các tài nguyên tham khảo quan trọng của Pano2VR.
  - Biết khi nào cần dùng tới `Direct Node Access`, `HTML template`, hoặc `JavaScript API`.

## 2. Một số chủ đề nâng cao nên biết

### 2.1. Direct Node Access (DNA)

- Cho phép truy cập trực tiếp vào một node cụ thể.
- Hữu ích khi:
  - Muốn chia sẻ link thẳng đến một điểm trong tour.
  - Gắn tour với hệ thống ngoài.

### 2.2. HTML Template

- Dùng khi cần tùy biến sâu lớp HTML bao quanh tour.
- Phù hợp với:
  - Dự án tích hợp website riêng.
  - Cần chèn tracking, branding, hoặc khối nội dung tùy biến.

### 2.3. JavaScript API

- Dùng để điều khiển tour từ mã JavaScript.
- Ví dụ:
  - Chuyển node bằng nút ngoài skin.
  - Đồng bộ tour với giao diện web riêng.
  - Tạo tương tác tùy biến nâng cao.

## 3. Khi nào mới nên đi sâu các chủ đề này?

- Khi workflow dựng tour chuẩn đã ổn.
- Khi dự án cần tích hợp vào hệ thống web/app lớn hơn.
- Khi skin mặc định không còn đáp ứng đủ nhu cầu.

## 4. Phím tắt và reference

- Ngoài docs theo từng chủ đề, phần `Reference` rất quan trọng để tra cứu nhanh:
  - Keyboard shortcuts
  - Thuộc tính
  - Các cấu hình output

## 5. Cách học nâng cao hiệu quả

1. Làm tốt workflow cơ bản trước.
2. Chỉ mở rộng khi có nhu cầu thật từ dự án.
3. Tạo các project thử nghiệm nhỏ để test tính năng nâng cao.
