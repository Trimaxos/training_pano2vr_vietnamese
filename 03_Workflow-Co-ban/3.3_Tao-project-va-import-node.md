# 3.3 – Tạo project và import node

## 1. Mục tiêu bài học

- Sau bài này, học viên sẽ:
  - Tạo được một project `.p2vr` mới đúng cách.
  - Import được ảnh panorama vào Tour Browser.
  - Hiểu các cách import khác nhau và khi nào nên dùng.

## 2. Tạo project mới

Các bước chuẩn:

1. Mở Pano2VR.
2. Chọn `New Project`.
3. Lưu ngay project bằng `Ctrl+S`.
4. Chọn đúng folder dự án, ví dụ:
   - `D:\VR360\DuAn_A\DuAn_A.p2vr`

> Không nên import ảnh trước rồi mới lưu project. Lưu sớm giúp đường dẫn tương đối ổn định hơn.

## 3. Các cách import node

### 3.1. Kéo thả vào Tour Browser

- Cách nhanh nhất và dùng nhiều nhất.
- Có thể kéo:
  - Một file panorama.
  - Nhiều file cùng lúc.
  - Cả folder.

Ưu điểm:
- Nhanh.
- Giữ nguyên cấu trúc làm việc trực quan.
- Folder name có thể trở thành tag tự động.

### 3.2. Dùng nút Add Input

- Phù hợp khi muốn chọn thủ công từng file.
- Thường dùng khi project nhỏ hoặc dữ liệu nằm ở nhiều nơi khác nhau.

### 3.3. Import lại hoặc thêm node vào project cũ

- Mở project hiện có.
- Kéo thêm file hoặc folder mới vào Tour Browser.
- Pano2VR sẽ thêm node mới mà không ảnh hưởng node cũ, trừ khi bạn xóa hoặc đổi liên kết thủ công.

## 4. Sau khi import sẽ có gì?

- Mỗi panorama trở thành một `node`.
- Thumbnail xuất hiện trong `Tour Browser`.
- Node đầu tiên thường trở thành `Start Node` mặc định.
- Tên node ban đầu có thể lấy từ tên file.

## 5. Việc nên làm ngay sau import

1. Kiểm tra node nào mở được bình thường.
2. Đặt `Title` trong `Properties → User Data`.
3. Kiểm tra đúng `Input Type`.
4. Xác định `Start Node`.
5. Sắp xếp lại thứ tự node nếu cần.

## 6. Import cả folder hay import từng file?

| Cách | Khi nên dùng |
|---|---|
| Import cả folder | Dự án có cấu trúc rõ ràng theo tầng/khu vực |
| Import từng file | Chỉ có vài node hoặc cần kiểm soát chặt từng ảnh |

Trong dự án thực tế, import theo folder thường hiệu quả hơn vì:
- Tận dụng tag tự động.
- Giảm thao tác tay.
- Dễ mở rộng khi thêm node sau này.

## 7. Lỗi hay gặp

| Lỗi | Nguyên nhân thường gặp | Cách xử lý |
|---|---|---|
| Missing file | Ảnh bị di chuyển sau khi import | Dùng `Find and Relocate Missing Files` |
| Ảnh méo | Sai input type | Chỉnh lại trong Properties |
| Node tên khó hiểu | Tên file nguồn không chuẩn | Sửa lại `Title` ngay sau import |
| Import xong quá lộn xộn | Chưa tổ chức folder trước đó | Sắp xếp lại source và import lại nếu cần |

1. Tạo một project mới và import ít nhất 6 ảnh từ 2 folder khác nhau.
