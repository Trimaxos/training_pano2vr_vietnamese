# 6.4 – Transformation Output

## 1. Mục tiêu bài học

- Sau bài này, học viên sẽ:
  - Hiểu Transformation Output dùng để làm gì.
  - Biết các tình huống nên dùng loại output này.

## 2. Transformation Output là gì?

- Dùng để xuất panorama thành:
  - Ảnh góc nhìn cố định.
  - Little planet / tiny planet.
  - Các biến thể hình ảnh khác phục vụ trình bày.

## 3. Khi nào nên dùng?

- Cần ảnh tĩnh từ panorama.
- Cần hình minh họa marketing.
- Cần tạo preview ngoài tour web.

## 4. Tài liệu tham khảo

- Transformation Output – `https://ggnome.com/doc/pano2vr/output/transformation/`

## 5. Gợi ý bài tập

1. Xuất một ảnh góc nhìn cố định từ một node.
2. Thử một bản tiny planet để so sánh.
