# 6.1 – Tổng quan các loại Output

## 1. Mục tiêu bài học

- Sau bài này, học viên sẽ:
  - Biết các loại output chính trong Pano2VR.
  - Chọn đúng loại output theo mục đích.

## 2. Các nhóm output chính

| Loại | Mục đích |
|---|---|
| Web | Xuất tour HTML5 để chạy trên browser |
| Transformation | Xuất ảnh hoặc góc nhìn tĩnh |
| Animation | Xuất video chuyển động |
| VR | Xuất tour cho môi trường VR/WebXR |
| Package/Cloud | Đóng gói hoặc triển khai nhanh |

## 3. Loại quan trọng nhất trong dự án thực tế

- Thường là `Web Output`.
- Đây là định dạng bàn giao phổ biến nhất cho khách hàng.

## 4. Tài liệu tham khảo

- Output Overview – `https://ggnome.com/doc/pano2vr/output/`

## 5. Gợi ý bài tập

1. Mở Output Panel và liệt kê những loại output bạn có thể thêm trong project hiện tại.
