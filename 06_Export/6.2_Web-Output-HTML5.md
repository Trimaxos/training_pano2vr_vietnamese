# 6.2 – Web Output (HTML5)

## 1. Mục tiêu bài học

- Sau bài này, học viên sẽ:
  - Tạo được một Web Output hoàn chỉnh.
  - Hiểu các thiết lập cơ bản khi xuất tour web.

## 2. Web Output dùng để làm gì?

- Xuất tour thành website HTML5 chạy trên trình duyệt.
- Đây là cách bàn giao phổ biến nhất.

## 3. Các bước tạo Web Output

1. Mở `Output Panel`.
2. Nhấn thêm output mới.
3. Chọn `Web`.
4. Chỉ định thư mục output.
5. Gán skin phù hợp.
6. Generate output.

## 4. Các thiết lập quan trọng

| Thiết lập | Ý nghĩa |
|---|---|
| Output folder | Nơi lưu file xuất |
| HTML file | Tên file khởi động |
| Skin | Giao diện dùng cho tour |
| Tiles | Thiết lập chia tile hình ảnh |

## 5. Lưu ý thực tế

- Dùng tên folder không dấu.
- Kiểm tra lại preview sau mỗi lần generate.
- Nếu giao diện không đúng, kiểm tra skin đã gán đúng chưa.

## 6. Tài liệu tham khảo

- Web Output – `https://ggnome.com/doc/pano2vr/output/web/`

## 7. Gợi ý bài tập

1. Tạo một Web Output cho project hiện tại.
2. Generate và mở preview bằng local server.
